# Panduan Pengguna - Newstep

Selamat datang di **Newstep**, platform terbaik untuk menemukan produk UMKM lokal berkualitas. Panduan ini akan membantu Anda menggunakan fitur-fitur yang tersedia di website kami.

## 1. Menjelajahi Produk

### Halaman Utama (Home)

Saat pertama kali membuka website (`index.html`), Anda akan disambut dengan produk-produk unggulan pilihan kami.

- **Hero Section:** Klik tombol **"Cek Produk"** untuk langsung menuju katalog lengkap.
- **Katalog Unggulan:** Scroll ke bawah untuk melihat 8 produk terbaru yang kami rekomendasikan.
- **Testimoni:** Baca pengalaman pelanggan lain di bagian bawah halaman.

### Halaman Katalog (Products)

Klik menu **"Products"** di navigasi atas untuk melihat semua koleksi kami.

- **Kartu Produk:** Setiap produk menampilkan gambar, nama, deskripsi singkat, dan harga.
- **Detail:** Klik tombol **"Lihat Detail"** (link simulasi) untuk informasi lebih lanjut.

## 2. Mencari dan Memfilter Produk

Fitur ini hanya tersedia di halaman **Products** (`main.html`).

- **Pencarian:** Ketik nama produk di kolom pencarian di sebelah kiri (desktop) atau tombol filter (mobile).
- **Kategori:** Pilih kategori tertentu (misalnya: _Makanan_, _Minuman_, _Kerajinan_) untuk menyaring produk.
- **Urutkan Harga:** Gunakan dropdown harga untuk mengurutkan produk dari termurah ke termahal atau sebaliknya.

## 3. Fitur Favorit

Simpan produk yang Anda sukai untuk dilihat nanti!

1.  **Menambahkan ke Favorit:**
    - Pada setiap kartu produk, klik tombol **"Hati"** (❤️).
    - Hati akan berubah warna menjadi merah, menandakan produk berhasil disimpan.
    - Sebuah notifikasi ("Toast") akan muncul di bawah layar mengonfirmasi tindakan Anda.
2.  **Melihat Favorit:**
    - Klik menu **"Favorite"** di navigasi atas.
    - Semua produk yang Anda simpan akan muncul di sini.
3.  **Menghapus dari Favorit:**
    - Klik kembali tombol hati merah pada produk di halaman mana saja, atau langsung di halaman Favorit.

## 4. Menghubungi Penjual / Admin

Jika Anda memiliki pertanyaan atau ingin bekerja sama:

1.  Buka halaman **"About"** (`about.html`).
2.  Gulir ke bagian bawah ke formulir **"Hubungi Kami"**.
3.  Isi **Nama**, **Email**, **Subjek**, dan **Pesan**.
4.  Klik **"Kirim Pesan"**.
5.  Tunggu notifikasi sukses yang muncul di layar.

## 5. Pertanyaan Umum (FAQ)

**Q: Apakah saya perlu mendaftar akun?**
A: Tidak, Anda bisa menjelajahi dan menyimpan favorit tanpa perlu mendaftar saat ini.

**Q: Bagaimana cara pembayarannya?**
A: Saat ini website berupa katalog. Untuk pembelian, silakan hubungi kontak yang tertera di halaman About.
